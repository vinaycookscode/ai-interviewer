# Deploy to Preview

Deploy current changes to Vercel preview environment.

## Steps

1. Ensure all changes are committed
// turbo
2. Run build to verify no errors:
   ```bash
   npm run build
   ```
// turbo
3. Push to develop branch:
   ```bash
   git push origin develop
   ```

4. Vercel will automatically deploy from develop branch

## Notes
- Preview URL will be generated by Vercel
- Check the Vercel dashboard for deployment status
